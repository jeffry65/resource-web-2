# Nissan Z Archive Prototype

Open `index.html` to preview locally.

## GitHub Pages upload
1. Create or open your repository.
2. Upload every file and folder from this package.
3. Make sure `index.html` stays in the root.
4. In repository settings, enable GitHub Pages from the main branch root.

## Included pages
- Home
- Models
- Archive
- About
- Individual model pages in `/models`
